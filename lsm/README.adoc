A Rust implementation of Liquid State Machines (spiking neural network reservoirs) for temporal pattern recognition.

== Overview

LSM implements reservoir computing using spiking neural networks for processing temporal/sequential data.

== Features

* Spiking neural network reservoirs
* Temporal pattern recognition
* Rust performance

== Installation

[source,bash]
----
cargo add lsm
----
