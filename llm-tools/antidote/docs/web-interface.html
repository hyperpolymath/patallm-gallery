How to Use Reset Artifacts

  1. Select an Artifact

    Choose the appropriate reset type for your needs. Use specialized resets (coding, conversation) when possible for better results.

  2. Copy the Artifact

    Click "Copy to Clipboard" or manually select and copy the artifact code.

  3. Paste into LLM Chat

    Paste the entire artifact into your LLM conversation (Claude, GPT, Gemini, Copilot, etc.)

  4. Wait for Confirmation

    The LLM should respond with "Reset complete" or a similar confirmation.

  5. Verify (Optional)

    Use diagnostic tools to verify the reset was effective.

  6. Continue Conversation

    Resume your conversation with a clean context slate.

Best Practices

  • Use specialized resets when applicable
  • Apply reset early rather than after deep context
  • Verify reset success with probe questions
  • Don't rely on resets for security-critical context clearing
  • Combine with preservation artifacts to maintain important context

About LLM Antidote

LLM Antidote is a collection of tools and artifacts designed to manage, reset, and manipulate language model context across multiple platforms. The project explores the boundaries of LLM behavior through minimal, symbolic interventions.

Philosophy

  • Minimalism: Artifacts should be simple and self-contained
  • User Agency: Tools empower users to control their LLM interactions
  • Cross-Platform: Solutions work across multiple model providers
  • Educational Focus: Advance understanding of LLM behavior

Supported Models

Reset artifacts have been tested with:

  • Claude (Anthropic) - 88-92% effectiveness
  • GPT-4 (OpenAI) - 75-78% effectiveness
  • Gemini (Google) - 70-75% effectiveness
  • GitHub Copilot - 65-82% effectiveness (context-dependent)
  • Mistral - 72% effectiveness
