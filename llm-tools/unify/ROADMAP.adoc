== Project Overview

LLM Unify is a production-ready Rust application providing unified access to AI conversation history across multiple platforms (ChatGPT, Claude, Gemini, Copilot). Built on principles of user autonomy, privacy-by-design, and local-first data ownership.

=== Current State (v0.1.0)

[cols="1,3"]
|===
|Component|Status

|**Architecture**
|6-crate Rust workspace with clear separation of concerns

|**Core Types**
|`Provider` enum, `Conversation`/`Message` structs, typed metadata

|**Storage**
|SQLite with WAL mode, FTS5 full-text search, connection pooling, schema versioning

|**Parsers**
|All four major providers: ChatGPT, Claude, Gemini, GitHub Copilot

|**CLI**
|14 commands: import, list, show, search, delete, export, stats, validate, backup, restore, init, schema, tui, version

|**TUI**
|Ratatui-based browser with vim bindings, search mode

|**Backup/Restore**
|SHA-256 checksums, atomic operations, integrity verification

|**Security**
|Zero unsafe code, parameterized queries, input validation

|**Compliance**
|RSR Silver (51/55), AGPL-3.0 + Palimpsest, RFC 9116
|===

== Appendix

=== Version Numbering

Following Semantic Versioning (SemVer):

* **MAJOR** (1.x.x): Breaking API changes
* **MINOR** (x.1.x): New features, backward compatible
* **PATCH** (x.x.1): Bug fixes only
